# Estructura del Proyecto CoPHo

## Organización de Directorios

```
CoPHo/
├── src/                    # Código fuente principal
├── configs/                # Configuraciones Hydra
├── data/                   # Datasets (incluyendo DIMACS)
├── input_data/             # Datos de entrada adicionales
├── scripts/                # Scripts de entrenamiento y utilidad
├── tools/                  # Herramientas de análisis y visualización
├── docs/                   # Documentación del proyecto
├── setup.py                # Configuración del paquete
├── requirements.txt        # Dependencias
└── .structure.md           # Este archivo
```

## Contenido de Carpetas

### `scripts/`
- `run_all.sh` - Ejecuta todo el pipeline
- `run_condi_pipeline.py` - Pipeline con condiciones
- `train_dimacs.sh` - Entrenamiento DIMACS
- `generate_graphs.sh` - Generación de grafos

### `tools/`
- `evaluation.py` - Evaluación de modelos
- `visualize_graphs.py` - Visualización de grafos
- `plot_generated_graphs.py` - Generación de plots individuales

### `docs/`
- `README.md` - Documentación principal
- `READY_TO_TRAIN.md` - Guía de preparación
- `SETUP_COMPLETE.md` - Información de setup
- `LICENSE` - Licencia del proyecto

## Cómo Ejecutar Scripts

Desde la raíz del proyecto:
```bash
# Ejecutar scripts
bash scripts/run_all.sh
bash scripts/train_dimacs.sh
python tools/plot_generated_graphs.py

# O navegar a scripts/
cd scripts
bash run_all.sh
```
# entrenar a mano
```bash
conda activate copho
cd src
python main.py --config-name=config_dimacs general.name=dimacs_exp1
```
